Sobre mi

Soy David Cruz, un entusiasta de la informatica y todo de esta. Esto es solo un mostrador de mis proyectos y una presentación de mi persona. Debajo dejo una lista de conocimientos sobre el campo de la informatica que creo podrian interesar como también mi nivel con estos.

 

Habilidades

Programación - Básico

  • Python - Tengo nivel de programación basico (Impartido en el instituto), se usar matrices, listas, diccionarios y usar documentos de este.
  • PHP - He realizado formularios que tratan la información transmitida hacia nuestra base de datos y el escape de esta para evitar ataques.
  • Bash Scripting - Hice una papelera para un ubuntu server, utilizaba diversos comandos, creacion de logs, restauracion de archivos (como la papelera de Windows) y que cada cierto tiempo, esta fuera vaciando los archivos que no habian sido utilizados en los anteriores 2 meses.

 

 

Administración de Sistemas - Intermedio

  • DHCP - Creación de un servidor y aplicación de diversas configuraciones sobre estos en RedHat, Debian y Windows; y como proteger este servidor, contra de DHCP Spoofing y DHCP Sniffing.
  • DNS - Creación de un servidor y configuración de este en RedHat, Debian y Windows. Y proteccion contra DNS Spoofing.
  • FTP / SFTP - Creación de un servidor de ficheros para que se pueda acceder desde una pagina web o para que se pueda usar desde la linea de comandos, configurando permisos y accesibilidad.
  • Servidor Web - Creacion de servidores web con diferentes configuracion como tambien balanceo de carga. He usado NGINX, Apache2 y IIS.
  • Clusters - Creación de un cluster de alta disponiblidad y configurado los nodos de estos.
  • Raids - Creación de discos que usaba conjuntamente diversos tipos de Raid ( 0, 1, 5, 10).
  • Virtualización - Virtualización local (VirtualBox y VMWare) y un servidor de virtualización con 3 maquinas. Y creación de maquinas virtuales y contenedores en esta.
  • Active Directory - Simulación de un Active Directory legal, con creación massiva de usuarios y configuración de permisos.
